# Task: WillowOps Targeted Prototype Build Direction

Choose the single best targeted prototype to build next as a follow-up work
sample for Willow Grey Interiors.

Use only the evidence packet. Do not ask to inspect the repo. Do not write code.
Do not propose a full redesign or a broad operating platform.

Context:
- Willow Grey has many disconnected tools and manual admin handoffs.
- The buyer is considering agencies and independent contractors.
- The strongest follow-up should make the independent-contractor advantage feel
  concrete: fast, focused, commercially useful.
- The prototype should show one simple business problem being solved by
  connected systems, not a broad internal platform.

Decision question:

What is the single best targeted prototype to build next so the follow-up feels
credible, simple, and directly tied to Willow Grey's pain?

Evaluate these options:
1. Enquiry-to-discovery brief
2. Project status update across Monday / Outlook / Studio Designer
3. Supplier delay communication workflow
4. Weekly leadership visibility dashboard
5. Another narrower option, if clearly better

Required output, in this order:

Recommendation:
- Pick exactly one prototype direction in 1-2 sentences.
- Explain why it is the strongest follow-up.

User story:
- Exactly 3 bullets.
- One sentence per bullet.
- Describe what Lucy or the team would see happen.

Systems shown:
- List only the tools that need to appear in the demo.
- For each, state its visible role in plain English.

Prototype scope:
- What should be on the page.
- What should be hidden behind technical proof.
- What should not be included yet.

Copy direction:
- Suggest the headline.
- Suggest one short supporting paragraph.
- Suggest labels for the primary demo steps.

Risk check:
- Name anything that could feel overclaimed, too technical, or too complex.

Build handoff:
- Exactly 4 bullets.
- Refer only to packet file/function names where relevant.
- Keep each bullet narrow enough to complete in one editing pass.

Follow-up message angle:
- Give 2 concise bullets for how to frame the prototype in a follow-up email.

Missing evidence:
- List at most 3 items.
- If none, say `None material from this packet.`

Budget:
- Target 900-1,100 words.
- Complete every section before adding detail.
- Do not end mid-sentence.
