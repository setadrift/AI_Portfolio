# Willow Grey Interiors

## Phase 1: Systems and Workflow Discovery

Prepared by Duncan Anderson
10 July 2026

## Purpose

Willow Grey has a clear opportunity to reduce repeated administration and make Monday.com, Studio Designer, and the supporting tools work together more effectively. Before changing live systems or building automations, we should agree the simplest way the business needs to run.

This short discovery phase will give Caroline and Lucy a practical, shared plan for doing that.

## Why Start With Discovery

Willow Grey's processes currently span Monday.com, EstiMAC, Studio Designer, email, documents, Xero, and team conversations. The business is also moving towards pod-based delivery while introducing Studio Designer.

Starting implementation immediately would risk automating steps that should first be simplified, connecting the wrong systems, or creating work that has to be redone during the Studio Designer transition.

Discovery will allow us to:

- remove unnecessary steps before automating them
- make a clear decision about what should be managed in each system
- protect the live systems the team already depends on
- focus the first investment on the highest-value improvement
- give the first implementation project a clear scope, cost, and measure of success

The result will be less rework, lower implementation risk, and a plan that makes sense to the people who use it every day.

## Scope

Over up to four focused consulting days, spread across two calendar weeks, I will:

- review the live Monday.com workspace, key documents, and representative projects
- run up to four short stakeholder sessions using real project examples
- map the workflow from enquiry through design, procurement, installation, and closeout
- identify duplicate entry, manual chasing, unclear ownership, and reporting gaps
- agree the role of Monday.com, Studio Designer, Xero, and the supporting tools
- define the first implementation pilot in enough detail to scope and quote separately

## Deliverables

At the end of Phase 1, Willow Grey will have:

1. A concise map of the current workflow and its main friction points.
2. A review of the Monday.com setup, including duplication and reporting gaps.
3. A plain-English recommendation for what should live in each system, when it should be updated, and who should own it.
4. A prioritized plan covering immediate no-regret improvements and a 90-day roadmap.
5. A clear brief for the first implementation pilot, including the intended outcome, safeguards, success measures, and estimated effort.
6. A findings review with Caroline and Lucy, followed by one consolidated revision so there is a single agreed direction.
